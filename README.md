# 🛒 Nxt Trendz – Cart Features

🚀 A React-based e-commerce application implementing advanced cart functionalities like quantity management, item removal, and dynamic price calculation.

---

## 🎥 Demo
https://assets.ccbp.in/frontend/content/react-js/nxt-trendz-cart-features-output.mp4

---

## 🌟 Features
- Add products to cart  
- Increase / Decrease item quantity  
- Remove items from cart  
- Dynamic total price calculation  
- Cart summary (items & total)  
- User authentication (Prime & Non-Prime)  
- Responsive UI
